## Development

You must do 

```
$ git clone <THIS_REPOSITORY>
$ cd <THIS_REPOSITORY_NAME>
```

### local

#### pakcage install

```
$ npm i
```


#### scss compile (watch)

```
$ npm run scss
```

#### add prefix

```
$ npm run prefix
```

#### scss compile and add prefix (watch)

```
$ npm run test
```
